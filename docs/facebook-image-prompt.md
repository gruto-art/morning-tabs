# Facebook post image prompt

Drop this prompt into Midjourney, DALL·E 3, Imagen, or any modern image
generator. Aspect ratio 1.91:1 (1200×630) is ideal for Facebook link previews.

---

## Primary prompt

> A clean, modern desktop illustration in a warm sunrise color palette
> (peach, soft orange, cream, deep navy accents). Centered: a stylized Mac
> laptop seen from above on a wooden desk. The laptop screen displays a
> minimal app window with a single large green button labeled "Open & Next"
> and a short vertical list of website favicons (Gmail, Instagram,
> Facebook, GitHub, calendar) lined up beside check-marks. Floating gently
> above the laptop, a small bouquet of organized browser tabs in different
> pastel colors — each tab tagged with a tiny chip showing a different user
> profile silhouette, suggesting "multiple accounts handled cleanly". Soft
> morning light streams in from the upper left, casting a long warm glow.
> A small ☕ coffee cup and a notebook sit beside the laptop. Visual mood:
> calm, intentional, productive, the opposite of overwhelm. Flat
> illustration with subtle gradient shading, fine line work, no text
> beyond the button label. 1.91:1 aspect ratio, generous negative space
> for adding marketing copy. Style references: Dropbox illustrations,
> Stripe marketing art, Notion onboarding screens.

## Negative prompt (for SD / Imagen)

> chaotic, cluttered, busy, photorealistic, stock-photo, generic office
> photo, dark, neon, cyberpunk, anime, hands, faces, text in random
> places, logos of real companies, watermark, harsh shadows, low contrast,
> cropped subject

## Quick variants if the first take is off

- **More minimal**: append "ultra minimal, single laptop, only the button
  visible, lots of empty cream background"
- **More playful**: append "tabs as small paper airplanes flying in a
  neat line into the laptop"
- **More productivity-app vibe**: append "isometric perspective, slight 3D,
  Behance hero illustration style"
- **Square version for Instagram**: change to "1:1 aspect ratio" and pull
  the laptop slightly smaller so the bouquet of tabs sits above it
- **Story / Reels version**: "9:16 vertical, the laptop centered lower,
  the tabs cascading up the frame"
